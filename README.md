# Top-Skills-for-Data-Analyst

In this project, I have used a dataset from Kaggle that pulls data from different online sites daily, to do EDA using Python to find the top 5 skills for a Data Analysts.

**Tool Used**:
* Jupyter Notebook

**Libraries Used**:
* Pandas
* Numpy
* Matplotlib
* Seaborn
* ast

**Techniques Used**:
* Data Cleaning
* Data Reshaping
* Feature Engineering
